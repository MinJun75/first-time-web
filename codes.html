Basic rules

Codes


<title></title>-사용자에게 보여지는 웹페이지의 이름을 지정하는 태그
<meta charset="utf-8">-한글을 인식하게해주는 태그
<br>-한 단락 띄우는 태그
<p></p>-하나의 문단으로 인식시켜 다른 글과 한 단락씩 띄우는 태그
<h1~6></h1~6>-머릿말을 나타내는 태그 (1~6순으로 숫자크기가 작아짐)
<img src="이름.jpg" width="크기">-사진을 표기하는 태그
<a href="링크 주소"></a>-지정 구간까지를 하이퍼링크를 거는 태그
<li></li>-목차 쓸 때 쓰는 태그
<ul></ul>-<li></li>를 묶을 때 쓰는 태그
<ol></ol>-<li></li>를 묶으면서 자동 순번처리해주는 태그
